# 🏨 BookBnB – Full-Stack Room Booking Platform

An end-to-end MERN stack booking application featuring secure user management, dynamic property listings, and robust workflows.

## 🛠️ Tech Stack & Key Implementations
- **Frontend:** React.js for dynamic UI rendering and responsive state management.
- **Backend:** Node.js & Express.js architecture for high-performance REST API handling.
- **Database:** MongoDB for seamless schema modeling and optimized cloud querying.
- **Security:** JSON Web Tokens (JWT) for secure authentication and strict role-based access control (Hosts/Guests).
- **Media Management:** Integrated Cloudinary API for optimized image storage and 30% faster asset loading.

The codebase is currently undergoing security patching and database query refactoring inside the local testing branch to align with modern corporate standards. Core framework production commits incoming.
